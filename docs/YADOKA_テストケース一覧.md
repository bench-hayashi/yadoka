## Claude Codeへの変換指示（コピペ用）

```
docs/YADOKA_テストケース一覧.md のテストケースをPlaywrightの自動テストに変換してください。

■ 進め方
1. 各テストケースを確認
2. 自動化可能なもの（○および△の操作確認部分）をPlaywrightテストに変換
3. 目視判定が必要なもの（レイアウトの崩れ、画像の見やすさ等）は
   skip + コメントで「手動確認項目」として残す
4. テスト名にNo.を含める（例：test('T-04: エリアから検索できる', ...)）

■ ファイル構成
- tests/e2e/traveler.spec.ts（T-xx）
- tests/e2e/owner.spec.ts（O-xx）
- tests/e2e/admin.spec.ts（A-xx）
- tests/e2e/common.spec.ts（C-xx）
- 既存テストと重複するものは統合・整理

■ ログインが必要なテストは helpers/auth.ts のヘルパーを使用

変換できた項目数と、目視確認が必要な項目をリストアップして報告してください。
```
